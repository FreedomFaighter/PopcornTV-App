Step 1: Change Apple TV DNS

In this step we change the "DNS" server for your Apple TV to point to PopcornTV. This redirects all traffic from the "Trailers" application to the PopcornTV App.

  1. Go into the "Settings" Application on your Apple TV
  2. Select "General"
  3. Select "Network"
  4. Select your current network connection from the list
  5. Keep selecting until you recieve a screen similar to the one below this
  6. Select "Configure DNS" then set it to "Manual"
  7. Select "DNS" and set it to then press "Done"
  8. Exit back to the "General" page

Once you have finished changing the DNS please select the "Next" Button
